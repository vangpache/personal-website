Bookit List

Bookit List is a full CRUD application created to simplify the planning of book study groups. Anyone with a registered account can create a reading circle specific to the book they are interested in reading and invite their friends to join the circle. Bookit List is a creative tool to inspire and promote a love for reading and learning. In this project I used, React, React-Redux, Redux-Saga Material UI, and the 3rd party API Good Reads.

Contribute Woodbury

Contribute Woodbury is a group project for our client Woodbury Community Foundation. This application is full CRUD and serves as a centralized hub for local nonprofits to advertise their services and events to the community. Community members can also utilize the application to seek out volunteer opportunities with the local nonprofits and engage in the local happenings. As a member of this project, I helped build the application from an idea, to creating wireframes and consulting with user experience designers, to its full functionality on both the client and server side.

Gallery of My Life

Gallery of My Life is a fun, social application where I can post photos and share memories associated with each photo. I learned how to use conditional rendering to showcase a photo and the memory associated with it, upon clicking. I used a React framework and bootstrap along with css to style this application.
